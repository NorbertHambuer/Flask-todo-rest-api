This is a rest api created with Flask!
Api methods
Method URL Action
GET https://flask-todo-rest-api.herokuapp.com/tasks Retrieve list of tasks
GET https://flask-todo-rest-api.herokuapp.com/tasks?id={task_id} Retrieve a task
POST https://flask-todo-rest-api.herokuapp.com/tasks Create a new task
PUT https://flask-todo-rest-api.herokuapp.com/tasks?id={task_id} Update an existing task
DELETE https://flask-todo-rest-api.herokuapp.com/tasks?id={task_id} Delete a task

POST fields:
  • description(string)
  • deadline(date)
  • dompleted(boolean) - optional
PUT fields:
  • id(integer)
  • description(string) - optional
  • deadline(date) - optional
  • dompleted(boolean) - optional
